## Image Encryption Application

A secure image encryption system using cryptographically secure pseudo-random number generators.

**APP link:**  
https://image-encryption-app-fwafojnczmca2m65xvqsjh.streamlit.app/

**Features:**
- CSPRNG based encryption (LCG & BBS)
- Pixel-level scrambling
- Encryption quality analysis using:
  - MSE
  - RMSE
  - Entropy
- Secure image upload & encryption via web UI

**Tech Stack:**
- Python
- Flask
- OpenCV
- NumPy
- Cryptography

**Use Case:**
Ideal for secure image transmission, academic cryptography projects, and cybersecurity demonstrations.
